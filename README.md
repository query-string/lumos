## Installation

Add this line to your application's Gemfile:

```ruby
gem 'lumos'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install lumos

## Devider

**Single call without params**
```ruby
lumos
```

**Specify delimiter**
```ruby
lumos :>, "->"
```

**Specify number of repeats**
```ruby
lumos :>, "@", 10
```
